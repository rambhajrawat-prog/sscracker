Delhi Police – Number & Alphabet Series / श्रेणी

🎙 Concept / आधार

Series में एक logic होता है — बढ़ोतरी, घटाव, गुणा, जोड़, position rule etc. हमें missing term या wrong term find करना होता है।

Types of Series / प्रकार

  • Number Series (संख्या श्रेणी)
  • Alphabet Series (अक्षर श्रेणी)
  • Mixed Series
  • Skip Series
  • Pattern Series

✔ Solved Examples – Number Series

  1. 2, 4, 6, 8, ? Ans: 10 ( +2 pattern )
  2. 5, 10, 20, 40, ? Ans: 80 ( ×2 pattern )
  3. 3, 9, 27, ? Ans: 81 ( ×3 pattern )
  4. 11, 14, 17, 20, ? Ans: 23 ( +3 pattern )
  5. 90, 80, 70, 60, ? Ans: 50 ( −10 pattern )

✔ Solved Examples – Alphabet Series

  1. A, C, E, G, ? Ans: I ( +2 rule )
  2. B, E, H, K, ? Ans: N ( +3 rule )
  3. Z, X, V, T, ? Ans: R ( −2 rule )
  4. A, D, G, J, ? Ans: M ( +3 rule )
  5. M, N, O, P, ? Ans: Q ( +1 rule )
