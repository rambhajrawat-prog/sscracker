Delhi Police – Figure Series / आकृति श्रेणी

🎙 Introduction / परिचय

Figure Series में एक आकृति दूसरी आकृति में किस logic से बदल रही है, यह पहचानना होता है। Delhi Police में केवल आसान patterns पूछे जाते हैं:

  • Rotation (घुमाव)
  • Flip / Mirror (उल्टा‑सीधा)
  • Shading change (रंग/शेडिंग)
  • Element add/remove (आकृति जोड़ना/घटाना)
  • Movement (स्थिति बदलना)

✔ Common Patterns / सामान्य logic

  • 90° clockwise rotation
  • 90° anti‑clockwise rotation
  • Alternate shading
  • Element shifting left → right
  • Number of lines increasing
  • Dot position moving in sequence

✔ Solved Examples (Simple DP Level)

Example 1 – Rotation Pattern
⬆ → ➡ → ⬇ → ⬅ Next = ⬆ (cycle repeats)
Example 2 – Dot Movement
Dot moves clockwise in each frame:
Top → Right → Bottom → Left → ? Ans: Top
Example 3 – Shading Pattern
Shaded → Unshaded → Shaded → Unshaded → ? Ans: Shaded
Example 4 – Increasing Lines
1 line → 2 lines → 3 lines → ? Ans: 4 lines
Example 5 – Element Addition
○ → ○● → ○●○ → ? Ans: ○●○●

✔ Practice MCQs

  1. Triangle rotates 90° clockwise each time. Next position?
    Ans: Upright triangle
  2. Square shading alternates. Last unshaded → next?
    Ans: Shaded
  3. Dot moves right → bottom → left → top → ?.
    Ans: Right
  4. Line count: 2, 4, 6, 8, ?.
    Ans: 10
  5. Half‑circle flips orientation. Up → down → up → ?.
    Ans: Down
  6. Arrow rotates anti‑clockwise 45°. After 3 steps?
    Ans: 135° position
  7. Star expands: 4 points → 5 points → 6 points → ?.
    Ans: 7 points
  8. Shaded area shifts clockwise. Next?
    Ans: Right side shaded
  9. Line shifts upward each step. Next?
    Ans: Top position
  10. Cross divides into more sections (2→4→6→?).
    Ans: 8

🔥 Challenge Question

Circle → Circle with dot → Circle with two dots (opposite) → ?

Pattern: dots doubling symmetrically
Answer: Circle with 4 dots (90° apart)
