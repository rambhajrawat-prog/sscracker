Delhi Police – Missing Number / Missing Term

🎙 Concept / आधार

Missing Number questions में किसी pattern को पहचानकर missing value निकालनी होती है। Delhi Police में यह series, matrix या box-based simple logic से पूछा जाता है।

✔ Common Logic Patterns

  • Addition / जोड़
  • Subtraction / घटाव
  • Multiplication / गुणा
  • Square / Cube logic
  • Opposite numbers / cross addition
  • Row–Column total / multiplication

✔ Solved Examples (Delhi Police Pattern)

Example 1: 3, 6, 9, 12, ? Pattern: +3 Ans: 15
Example 2: 2, 5, 11, 23, ? Pattern: ×2 +1 Ans: 47
Example 3 (Matrix):
4 | 8 6 | ?
Pattern: second = first ×2 6 × 2 = 12
Example 4 (Cross Addition):
(3 + 4) = 7 (5 + 6) = 11 Missing for (7 + 8) = 15
Example 5 (Square Logic):
Next = previous number + (position²)
5, 6, 10, 19, ? Positions: +1, +4, +9 → next +16 Ans: 35
