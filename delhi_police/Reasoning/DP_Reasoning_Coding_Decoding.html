Delhi Police – Coding & Decoding / कूट लेखन

🎙 Concept / आधार

Coding–Decoding में एक शब्द/संख्या को किसी rule के अनुसार बदलकर लिखा जाता है। Exam में हमें उस rule को पहचानकर नए शब्द का code खोजना होता है।

Major patterns:

  • Letter shifting (+1, −1, +2 etc.)
  • Position values (A=1, B=2 … Z=26)
  • Opposite letters (A↔Z, B↔Y …)
  • Word coding (replace, reverse, rearrange)
  • Number coding (sum, product, square)

1️⃣ Letter Coding / अक्षर आधारित कोडिंग

Example: If CAT is coded as DBU, what is code for DOG?
C→D (+1), A→B (+1), T→U (+1)
Same rule:
D→E, O→P, G→H → EPH
Example: If FAN → EBM, what is PEN?
F→E (−1), A→B (+1), N→M (−1)
Use −1,+1,−1:
P→O, E→F, N→M → OFM

2️⃣ Word Coding / शब्द आधारित कोडिंग

Example: If TEACHER → TERCAHE, what is STUDENT?
Pattern: First 3 letters same, last 3 reversed.
STUDENT → STU + DNE → STUDNE
Example: If POLICE → ELICOP, what is DOCTOR?
Pattern: Reverse + shift.
DOCTOR → ROTCOD → ROTCOD

3️⃣ Number Coding / संख्या आधारित कोडिंग

Example: If HOT → 40, what is FUN?
H(8)+O(15)+T(20)=43 → −3 = 40
FUN: F(6)+U(21)+N(14)=41 → −3 = 38
Example: If DELHI → 38, find JAIPUR?
D(4)+E(5)+L(12)+H(8)+I(9)=38
J(10)+A(1)+I(9)+P(16)+U(21)+R(18)=75 → 75

4️⃣ Mixed Coding / मिश्रित कोडिंग

Example:
DOG → XTR
CAT → XPA
Find BAT?

Observe mapping:
D→X, O→T, G→R (predefined mapping set)
Similarly B→X, A→P, T→A → XPA

🔥 Challenge Question

If POLICE → QPMJDF, then CRIMINAL → ?

Shift +1 rule:
C→D, R→S, I→J, M→N, I→J, N→O, A→B, L→M
Answer: DSJNJOBM
