Chapter

Tables & Graphs (DI) / सारणी व आलेख (DI)

Data Interpretation basics for Delhi Police: table, bar, line, pie – with quick calculations.

Smart Tips / आसान ट्रिक्स

  • पहले टेबल/ग्राफ को 10 सेकंड में समझो: units, heading, total.
  • DI में 70% marks calculation से नहीं, सही reading से आते हैं.
  • Approximation use करो: exact calculation सिर्फ जब options बहुत close हों.
  • Percentage change = (New-Old)/Old × 100 — बार-बार पूछते हैं.
  • Average = Total/Count — DI में सबसे common trap.

Formula / Key Notes

1) Percentage = (Part/Whole) × 100
2) Percentage change = ((New - Old) / Old) × 100
3) Ratio → Fraction: a:b = a/(a+b), b/(a+b)
4) Average = Sum / n
5) Pie chart angle = (Value/Total) × 360°
6) Speed = Distance / Time

Practice Questions (10)

Q1. A pie chart shows A=90°, B=120°, C=150°. What is % share of B?
Answer: C
B share = 120/360 = 1/3 = 33.33% (Option C).
Q2. In a table, total sales = 500. Product X sales = 125. X is what % of total?
Answer: B
125/500 = 0.25 = 25%.
Q3. Bar graph shows Jan=40, Feb=50, Mar=60. What is average?
Answer: B
(40+50+60)/3 = 150/3 = 50.
Q4. If a value increases from 80 to 92, % increase is:
Answer: C
(92-80)/80 ×100 = 15% (Option C).
Q5. A table shows total students=200. Boys=120. Girls= ?
Answer: C
Girls = 200-120 = 80.
Q6. In a line chart, value drops from 150 to 120. % decrease is:
Answer: C
Decrease=30. 30/150×100=20% (Option C).
Q7. If average of 5 numbers is 18, their sum is:
Answer: C
Sum = 18×5 = 90.
Q8. Pie chart angle for a category is 72°. What % is it?
Answer: C
72/360 = 20%.
Q9. From a table: Distance=120 km, Time=3 hr. Speed is:
Answer: C
120/3 = 40 km/h.
Q10. If total=600 and part=90, part as percentage is:
Answer: B
90/600 = 15%.
