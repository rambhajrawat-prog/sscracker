🚀 AI Features Setup — 5 Minutes
Jai Shree Shyam Ji! यह guide follow करके सभी AI features activate हो जाएंगे। बस 4 simple steps हैं — कोई coding नहीं, सिर्फ copy-paste।
⏱ Total time: 5-10 minutes

1
Cloudflare Worker बनाएं — Free Backend
1.1
Cloudflare account बनाएं
Browser में जाएं: workers.cloudflare.com → Sign up करें (सिर्फ email चाहिए, credit card नहीं)।
workers.cloudflare.com ↗
1.2
New Worker बनाएं
Login के बाद → Workers & Pages → Create Application → Create Worker → Worker का नाम दें: sscracker-proxy → Deploy click करें।
1.3
Code paste करें
Edit Code click करें → सारा existing code delete करें → cloudflare_worker.js का पूरा content copy करके paste करें → Save and Deploy click करें।
💡 cloudflare_worker.js file आपके SSCracker_v2 folder में है। Notepad से खोलें, Ctrl+A से सब select करें, copy करें।
1.4
Worker URL copy करें
Deploy होने के बाद आपको एक URL मिलेगा जैसे:
https://sscracker-proxy.YOUR-NAME.workers.dev
यह URL copy करके रख लें — Step 3 में use होगा।
2
API Key Worker में Add करें
2.1
Anthropic API Key लें
Browser में जाएं: console.anthropic.com → Login → API Keys → Create Key → Key copy करें।
⚠️ API Key किसी को share मत करें। यह आपके पैसे से जुड़ा है।
2.2
Key को Worker Secret में Add करें
Cloudflare Dashboard → आपका Worker → Settings tab → Variables and Secrets → Add variable:
Variable name: ANTHROPIC_API_KEY Value: sk-ant-...आपकी key... Type: Secret ← ✅ यह जरूर select करें
→ Save click करें।
✅ Secret type select करने से key encrypt हो जाती है — कोई देख नहीं सकता।
3
SSCracker में URL Configure करें
3.1
ss_config.js खोलें
SSCracker_v2 folder → ss_config.js file → Notepad से open करें।
3.2
PROXY_URL replace करें
File में यह line ढूंढें:
PROXY_URL: 'https://YOUR-WORKER.workers.dev',
इसे अपने Worker URL से replace करें:
PROXY_URL: 'https://sscracker-proxy.YOUR-NAME.workers.dev',
File save करें (Ctrl+S)।
3.3
Razorpay Key भी add करें (Task 6)
उसी ss_config.js में:
RAZORPAY_KEY: 'rzp_live_YOUR-ACTUAL-KEY', /* dashboard.razorpay.com → Settings → API Keys */
4
Test करें — सब काम कर रहा है?
4.1
Worker URL Test करें
अपना Worker URL नीचे paste करें और Test button दबाएं:
4.2
ask_doubt.html खोलें और test करें
Browser में ask_doubt.html open करें → कोई doubt submit करें → AI answer आना चाहिए।
✅ अगर answer stream हो रहा है — सब कुछ सही है! सभी 8 AI features अब काम कर रहे हैं।
